# Tru Blu V2 (Local-Only)

This is a **local-only** operations portal for Tru Blu Container Crew built with:

- Next.js App Router
- TypeScript
- Tailwind CSS
- localStorage mock data

## Run locally

```bash
npm install
npm run dev
```

Then open:

- `http://localhost:3000/login`

## Notes

- No Firebase
- No external database
- No payments/payroll
- All data is in browser localStorage
